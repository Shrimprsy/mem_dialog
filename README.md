# mem\_dialog

Research on Memory Algorithms

# 三层记忆存储

工作记忆（原始对话触发后压缩），情节记忆（设定固定轮次后压缩生成一条摘要以及相关标签），语义记忆（抽取了用户，事实，偏好，特征）

\-压缩过程：

&#x20; 1. 将新增的对话片段（仅未压缩部分）调用 LLM 生成一段简洁摘要

&#x20; 2. 同时调用 LLM 抽取结构化元数据（人物、地点、情绪、关键事实、偏好等）

&#x20; 3. 摘要和原始对话文本存入 L2 情节记忆，并生成向量索引

&#x20; 4. 抽取的关键事实和偏好增量更新到 L3 语义记忆库（相同 key 自动合并置信度）

&#x20; 5. 更新压缩进度指针，工作记忆本身不清空（仅标记已压缩），保证原始对话可随时回溯

# 记忆检索

每次用户发送新消息时:

从三个维度检索相关记忆，并加权融合排序

1.候选集初筛（元数据过滤）

利用 L2 情节记忆的标签（人物、地点、情绪、主题等）快速过滤候选记忆
2.dense+bm25
3.时间衰减因子
